                       DIFFERENCES BETWEEN NULL AND UNDEFINED
NULL 	                                              					 UNDEFINED
It is an object.          								It is a type undefined.
It is an assigned value.								It is not assigned.
In arithmetic conversion done on null it gives the value determined as zero.		Doesn’t carry out an arithmetic conversion. 
The value for variable is absent.							The variable itself is absent.
It is a primitive value.								It is a global variable.
